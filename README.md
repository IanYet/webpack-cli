# webpack-cli

---

一个简单的webpack demo项目

主要作用是备份一个打开即用的、无需配置的webpack项目模板，附带有一些常用的功能，以便下次使用

### 启动

```
npm install
npm run dev //开发模式
npm run build //编译
```

### 功能

- babel
- css-loader, style-loader
- file-loader, url-loader
- HMR
- devServer 404返回非index.html
- devtool模式为source-map

### 开发环境

- npm > 5
- node: 6.12.0
